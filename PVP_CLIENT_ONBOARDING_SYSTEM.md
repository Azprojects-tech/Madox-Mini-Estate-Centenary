## 📋 TEMPLATE STRUCTURE

### Core Template Files
1. **`PVP_TEMPLATE_PORTAL.html`** - Master template with placeholders
2. **`template-config.json`** - Configuration template
3. **`CLIENT_TEMPLATE_SHEET.csv`** - Google Sheets template structure
4. **`DEPLOYMENT_TEMPLATE/`** - Ready-to-deploy folder structure

### Placeholder System
**Template uses dynamic placeholders:**
```
{{ESTATE_NAME}} → "Greenfield Gardens Estate"
{{CLIENT_NAME}} → "John Doe"  
{{CLIENT_EMAIL}} → "john@greenfield-estates.com"
{{CLIENT_PHONE}} → "+234 801 234 5678"
{{GEOJSON_DATA}} → Client's parcel data
{{SHEET_ID}} → Generated Google Sheets ID
{{DEPLOYMENT_URL}} → Custom Netlify URL
```
